The sum is divisible by `k` if:

sum(nums) % k == 0

If the sum is not divisible, the smallest number of operations needed to fix the divisibility is exactly the **remainder** of this division.

---

## âœ”ï¸ Approach
1. Compute the sum of all elements.
2. Compute `sum % k`.
3. This remainder is the minimum operations needed.

public class Solution {

    // Function to calculate minimum operations
    public int minOperations(int[] nums, int k) {
        int sum = 0;

        for (int num : nums) {
            sum += num;
        }

        // Minimum operations needed is the remainder itself
        return sum % k;
    }

    // Main method for testing
    public static void main(String[] args) {
        Solution obj = new Solution();

        int[] nums = {1, 2, 3, 4};
        int k = 5;

        int result = obj.minOperations(nums, k);
        System.out.println("Minimum operations needed: " + result);
    }
}
